关于重构联赛体系、实施“省级行政区代表队”超级联赛顶层设计白皮书

发布日期: 2026-05-10

[toc] **呈报单位：** （虚拟） **日期：** 2026年5月 **主旨：** 彻底打破现有僵化体制，构建以34个省级行政区为参赛单位，以全民动员、财务透明、均摊反哺为核心原则的足球联赛新秩序。 --- ## 一、当前困境与改革的必要性 中国足球长期积弱的根本原因，并非人种或体能，而是**制度性的群众参与阻断**与**资源分配的严重悬浮**。 1. **注册球员基数崩塌**：现行体系依赖极少数职业俱乐部与足校，覆盖人口不足百万，导致14亿人中的绝大多数从未进入过选拔视野，天才被系统性埋没。 2. **地域认同感缺失**：俱乐部频繁搬迁、改名，沦为资本玩具，丧失了“为家乡而战”的荣誉根基，球迷文化成为无根之萍。 3. **市场与行政错位**：职业联赛既不商业，也不职业。权力集中于足协，规则制定随意，财务密室化，彻底丧失了公信力。 4. **家长缺乏信任**：因整个体系不透明，充斥潜规则与不确定风险，家长不敢将孩子送上绿茵场，导致人才输送的源头枯竭。 鉴于此，必须进行一次从底层逻辑到顶层架构的彻底革命。 ## 二、联赛架构设计 ### 1. 核心原则：一省一队，代表本土 逐步接替现有联赛赛制。以23个省、5个自治区、4个直辖市、2个特别行政区为参赛主体，每地组建一支一线队，共计34支球队，构成本方案的最高级别联赛。此项设计旨在强行以行政区划为锚点，将足球根系扎入每一寸国土。 ### 2. 赛程接轨：跨年制 联赛实行跨年制，与欧洲五大联赛及亚冠赛程同步。此举旨在打通与国际足坛的转会、租借窗口，降低球员留洋与引进外援的操作门槛。 ### 3. AB分组，精英与挑战并存 为解决34队单循环赛制的不公与冗长，将联赛分为A、B两组。 - **A组（16队）**：代表中国足球最高竞技与商业水平。 - **B组（18队）**：为挑战者组。 ### 4. 升降级制度：生存权倒逼，全员无安全区 低级别联赛为各省、自治区、直辖市、特别行政区内部的地方联赛。一线队的命运与省内联赛深度绑定，形成闭环压力系统。 #### 完整升降级路径 | 层级 | 球队范围 | 赛季末规则 | | -------- | -------------------- | ------------------------------------------------------------ | | A组 | 前12名 | 直接保级，留在A组 | | A组 | 后4名（顺位13-16） | 直接降入附加赛池，与本省省内联赛前3名、B组5-18名共同争夺次年联赛资格 | | B组 | 顺位1-4名 | **直接升级**，无需打附加赛，递补A组降级的4个名额，确保次年A组资格 | | B组 | 顺位5-18名（共14队） | 进入附加赛池，与本省前3名及A组降级队争夺次年留在全国联赛体系的资格 | | 省内联赛 | 前3名 | 进入附加赛池，与A组降级队及B组5-18名共同争夺次年联赛资格 | #### 流动通道总览 **升级路径（向上流动）：** 省内联赛前3名 → 附加赛 → B组 → B组常规赛前4名 → A组 **降级路径（向下流动）：** A组后4名 → 附加赛池 →（可能）降入B组或直接降至省内联赛 #### 设计深意 - **奖励B组头部，赋予常规赛极致价值**：B组前4名直接获得次年A组资格，确保整个B组常规赛每一轮都充满意义。争夺前四就是争夺“免死金牌”和直升A组的门票，彻底消除垃圾比赛。 - **制造精准的“危险区”**：B组5-18名（14队）+ A组降级4队 = 18队，与本省前3名共同争夺次年联赛席位。没有安全区，没有混日子的空间。 - **A组降级队的二次惩罚**：A组后4名不仅失去A组资格，还须立刻投身附加赛生死战，没有任何喘息空间。 - **省内联赛通道全面打开**：省内联赛前3名获得的是真正的升级通道，一支省内豪强可以一步登天进入全国联赛。这是对基层足球最强有力的激励。 ### 5. 完全开放的人才流通：废除户籍壁垒 坚决不设任何地方户籍限制，让“省队”成为开放的竞技平台，代表本省而战，而非仅由本省人组成。所有人都是中国人，都为了统一的目标而努力。这一设计用竞技压力倒逼出一个全国性的人才筛选与快速流通市场，让更多好苗子不被准入门槛所限制。 ### 6. 球员交易与自由转会体系 各级球队的球员买卖收入，是球队独立商业收入的重要组成部分，不进入公共池。完全开放的自由转会制度，将激活全国人才市场，使青训成为可持续的商业模式。 - **资产确权**：每支球队对其注册球员拥有完整的资产权利，转会费、培训补偿、联合机制补偿均归球队自身所有。 - **自由转会**：合同到期球员可自由加盟任何球队；合同期内转会由双方俱乐部协商转会费。 - **培训补偿与联合机制补偿**：严格执行国际足联标准，球员23岁之前每次转会，新俱乐部须向所有培养过该球员的俱乐部支付补偿金；合同期内转会，转会费的5%分配给球员12-23岁期间效力过的所有俱乐部。确保从基层村超到A组豪门的每一位培养者，都能在球员的成长和交易中获得经济回报。 - **交易透明**：所有转会合同、补偿金支付记录全部上链，公开可查。 - **青年球员保护**：禁止16岁以下球员跨省转会（可自由选择本省俱乐部）；18岁以下球员合同必须包含教育保障条款；设立“回购条款”，21岁以下球员出场时间未达标，母队有权以原转会费回购，防止年轻球员被囤积而荒废。 - **租借开放**：梯队球员可自由租借至其他省份任何级别联赛球队，租借期最长为两个赛季，母队不得设置阻碍。 此制度将迫使每支球队、每个梯队都成为精明的“球员开发商”，形成“挖掘 → 培养 → 使用 → 出售 → 再投资”的完整经营闭环，让中国足球的人才土壤因市场力量而真正肥沃起来。 ### 7. 鼓励球探公司、经纪公司等外围商业机构发展 人才市场的繁荣，需要专业化的中介服务作为润滑剂。本方案明确鼓励并依法规范以下外围商业机构的发展： - **球探公司**：为繁荣、拉通、丰富各级球队的信息资源，同时为各级球队提供球员数据采集、技术分析、潜力评估等专业服务，帮助球队降低选材的信息成本，提高青训挖掘效率。 - **经纪公司**：依法为球员提供合同谈判、转会协调、商业开发等服务，保障球员合法权益。 - **数据分析公司**：为球队提供比赛数据分析、对手情报、运动科学支持。 - **足球培训与咨询公司**：为教练员、裁判员、管理人员提供专业培训与技术咨询。 所有进入本体系运营的外围公司，须在联赛公司进行简易注册，接受行业自律规范，其重大服务合同须在区块链上进行存证，以确保市场秩序的透明与公平。联赛公司有义务为这些公司营造一个稳定、可预期的政策环境，让专业的人做专业的事，让市场机制在人才发现与培养中发挥最大的效率。 ## 三、渐进式改革路径：双轨并行，增量生长 ### 1. 核心策略：农村包围城市，以时间换空间 不在改革初期触碰现有中超、中甲、中乙体系的既有利益格局。不强制解散、不强制合并、不强制转让。让新体系从地方联赛和足球欠发达地区自然生长出来。 ### 2. 第一阶段（第1-2年）：地方联赛全面建设期 - **任务**：34个省级行政区全部建立省级联赛体系，向下覆盖地市级、县级联赛。 - **存量吸纳**：现有各类乙级联赛、业余联赛、村超、苏超等，全部以**注册制**自然并入所在省的省级联赛体系。不改变其名称、运营主体和赛制，只做身份注册和技术赋能，并设立五年“地方赛事保护期”。 - **基础建设**：联赛公司核心工作是统一球员注册系统、建立保险体系、培训地方裁判、提供赛事组织技术标准。**暂不举办全国性联赛。** - **旧体系**：足协在此期间继续管理中超、中甲、中乙，与新体系互不干涉。 ### 3. 第二阶段（第3-4年）：省队组建与B组启动 - **省队组建**：每省从省级联赛中选拔球员组建省队。选拔方式由各省自主决定。 - **B组启动**：34支省队开始B组联赛，这将是新体系的首次全国性联赛。 - **制度运转**：总资金池、均摊分红、工资红线税等财务制度随B组联赛同步启动。 - **窗口选择**：现有职业俱乐部可**自愿申请**将其一线队或预备队整体转化为所在省份的省队，前提是接受新体系的全部制度。不愿意的，继续留在旧体系。 ### 4. 第三阶段（第5年起）：A组诞生，双轨自然融合 - **A组诞生**：经过2-3个赛季的B组联赛，前16名自动组成A组。完整的AB组升降级制度正式启动。 - **市场选择**：旧体系球队将面临选择。留在旧体系，商业价值和关注度将随新体系的崛起而逐渐萎缩。加入新体系，必须接受透明的财务和公平的规则。**由市场替行政做出选择。** - **旧体系处置**：不禁止、不歧视，但不给予特殊保护政策。其球员可自由流动至新体系。 ## 四、商业与财务模型 此方案的根本逻辑是“**全国一盘棋，先富带后富，财务全上链**”。以此为绝对底色。 ### 1. 公共收入与绝对均摊 联赛公司的主要收入来源包括：**联赛冠名、全国转播权打包销售、以联赛为主体的赞助及各种联赛主体的广告收入**。此项收入为**公共池**，在赛季末，除公司基本运营费用外，**以绝对平均的形式（即各1/34）分红给34个地区的分公司**。不设任何成绩绩效权重。 各地区分公司依据年初制定的各个分公司的分红比例，分配给辖区内的各支球队。各项款项的划拨，均实时上链。 此设计确保了所有球队在起步阶段都能获得最基本的资金保障，让联赛的公平性从根基上得以保证。 ### 2. 全面纳入地方赛事收入 为彻底杜绝“各玩各的”、防止经济强省与弱省的收入差距在地方层面被再次拉开，所有地方赛事的商业收入，同样纳入全国公共池统一分配。具体包括： - 省内联赛的转播权销售收入、赛事主体赞助及广告收入。 - 地方特色赛事（村超、苏超等）的转播权销售收入、赛事主体赞助及广告收入。 - 附加赛产生的转播权及赞助收入。 - 地方赛事球队的工资红线税（如适用）。 以上各级赛事的商业收入，与全国联赛A组、B组的收入一并汇入全国公共池，赛季末按1/34平均分配。地方球队自身的球衣广告（排除联赛/杯赛标识区域）、主场门票、球队周边产品等收入，以及其他相关以球队为个体不包含整个联赛主体的各类广告、服务等，仍归球队自身所有。 （此设计的根本目的是让球队商务灵活起来，最大限度的开发球队的商业价值，充分为球队资金广开来路） ### 3. 引入“工资红线税”，取消硬工资帽 完全废除僵化的硬性工资帽制度，让商业的回归商业，让市场决定球员价值。但为防止资本无序扩张导致一家独大，设立**奢侈税红线**： - A、B组球队薪资总额超出红线的部分，须按超额累进税率向联赛公司缴税。 - 建议税率阶梯： - 超出红线0-20%部分征收1:1税款； - 超出20%-50%部分征收1:2税款； - 超出50%以上部分征收1:4重税。 - **税款全额注入公共池**，并在赛季末以均摊形式分红给其他33个地区分公司。各地区分公司依据年初制定的分红比例，分配给辖区内的各支球队。 此举实现了“**豪门越消费，联盟越富有**”的共赢逻辑，是市场化调节公平的利器，且与绝对均摊原则一以贯之。 ### 4. 财务收入汇总 赛季末，基于青训球员一线队出场时间、转会输出价值、国家队入选人次等指标，评分前5名的省份获得由“工资红线税”中专项拨付的青训奖励金，全部上链公开。 ## 五、权力与治理结构的重构 ### 足协：彻底转型为纯服务型公益机构 #### 一、总体定位 中国足协及各级地方足协全面剥离行政管控、赛事决策、行业管理等全部实权，从“行业管理者”彻底转型为“纯公共服务机构”。其职能实行清单化刚性划定，无清单外任何权限。 #### 二、核心服务职能（正面清单，仅限以下事项） - 球员统一注册、档案规范化管理及信息查询服务； - 各类足球赛事（含职业、业余）参赛资质审核、准入凭证签发； - 裁判员技术等级评定、资质认证及后续培训、考核服务； - 赛场判罚争议的中立评议、结果公示及解释说明； - 各级国字号球队（男足、女足各年龄段）的组建、集训及国际赛事组织协调； - 足球领域对外外事交流、国际合作对接及服务保障； - 足球运动基础普及、公益推广及基层足球公益项目落地。 #### 三、刚性约束（负面清单，明确禁止，无例外） - 无权制定、修改任何级别足球联赛的竞赛规则、赛制安排； - 无权插手联赛商业运营、招商合作、版权开发； - 无权任免、考核、奖惩联赛总公司、分公司及各俱乐部管理层人员； - 无权干预各俱乐部球员引援、薪资设定、赛程调整、内部处罚等一切联赛内部事务； - 不具备任何联赛层面的行政干预、指令下达权限，不得变相行使管理职能。 #### 四、运营经费与国字号球队费用（规范来源，独立核算） **1. 日常运营经费** 足协及各级地方足协的日常运营经费，由全国联赛总公司按双方商定的固定比例定期缴纳标准化公共服务费。经费具体任用事由由足协按服务职能统筹调配，调配明细同步上链公示。 **2. 国字号球队相关费用** 各级国字号球队的组建、集训及国际赛事组织协调等费用，由全国联赛总公司与足协按既定比例共同承担，单独列明开支明细。 **以上两项费用分开核算、分别上链存证**，与公益扶持资金严格区分，均接受社会、联赛公司及相关监管部门监督，确保专款专用、合规透明，缓解足协资金压力的同时兼顾联赛责任担当。 #### 五、公益反哺机制（跨项目帮扶，资金独立） 明确支持足协依托足球头部IP的商业影响力，开展跨体育项目公益赋能与战略帮扶，定向支持跳水、冰雪、举重、体操等奥运竞技项目（含国字号队伍及青训体系），实现体育资源全域统筹、良性反哺。 **资金管理要求**： 单独建立专项账户，实行专款专用、全程监管，资金收支即时上链存证、公开透明，绝不占用联赛运营资金、俱乐部发展资金及足协日常服务经费，确保全部用于跨项目公益帮扶。 ### 全国联赛总公司：地域共治 + 竞技实绩加权决策 全国划分 34 个省级行政主体，各主体推举专职行业代表，共同组建全国足球联赛总公司董事会，全权执掌全国联赛体系：赛事赛制制定、整体商业招商、全国性赛事版权统筹、联赛整体秩序准则、职业赛事顶层运营规划。 各省级区域同步设立联赛地方分公司，属地化处理本土赛事组织、基层联赛落地、区域赛事协调等属地事务，隶属总公司统筹管理。 建立**竞技实绩动态加权决策机制**，摒弃平均主义： 1. 上赛季顶级联赛综合战绩前五名的省级代表席位，自动持有董事会**一票专项否决权**； 2. 否决权限实行年度动态更迭，排名跌出前五即刻收回权限，由新晋前五地域代表顺位承接； 3. 权力核心逻辑：让实战成绩最优、运营理念最成熟的竞技强势方掌握核心决策话语权，倒逼联赛整体竞技水平、运营模式持续升级，杜绝外行主导内行决策。 董事会常规决策实行集体票决制，仅重大赛事改革、巨额商业合作、联赛架构调整等核心事项启用实绩席位否决权，兼顾集体公平与竞技专业导向。 ### 3. 俱乐部球队独立商业权益确权固化 明确划分**联赛公共商业权益**与**俱乐部自有私人商业权益**，权责清晰互不侵占： 1. 球队完整保留自主独立商业开发权限：非联赛统一指定区域球衣商业广告、球队主场自主票务运营、俱乐部专属文创周边、自建商业友谊赛、球队品牌个人代言合作、青训球员转会收益、青训培训补偿费用等； 2. 俱乐部自有商业全部营收，全额归属俱乐部自有账户，**不纳入联赛公共收益池统筹分配**； 3. 仅联赛统一赛事冠名、全国统一转播版权、联赛官方统一赞助商等全域公共商业收入，纳入联赛总公司公共资金池，用于联赛整体赛事投入、场地配套升级、基层足球普惠建设； 4. 从制度层面保障头部强队商业价值与盈利积极性，同时保障中小俱乐部依托联赛公共资源稳步生存，形成强弱良性循环的健康足球生态。 整体架构实现：**足协守服务底线、联赛掌运营实权、球队保自主收益、实绩定决策权重、足球 IP 反哺全域体育**，彻底理顺足球行业权责乱象，构建职业化、市场化、公平化、全域赋能的全新体育治理体系。 ## 六、裁判员体系：独立、透明、升降级 ### 1. 裁判公司化，彻底独立 成立**全国联赛裁判有限责任公司**，由联赛公司全资控股，与足协不参与管理。足协仅保留裁判员技术等级认证职能，裁判的选派、考核、处罚权完全归裁判公司。 ### 2. 三级裁判池 - **A级裁判池**：执法A组比赛，约40-50人，全部为全职职业裁判。 - **B级裁判池**：执法B组比赛，约60-70人，全职与兼职结合。 - **C级裁判池**：执法省内联赛及附加赛，由各省推荐，全国统一培训考核，兼职为主。 ### 3. 选拔机制：开放考试，逐级晋升 任何年满18岁的中国公民，均可报名参加年度裁判资格考试。通过者进入C级裁判池。C级裁判赛季末综合评分前5%者，获得B级晋升资格；B级裁判连续两个赛季综合评分前10%者，获得A级晋升资格。 ### 4. 淘汰机制：每场打分，末位淘汰 **三级裁判池**：A级（全职，执法A组）、B级（全职与兼职结合，执法B组）、C级（兼职，执法省内联赛及附加赛） - A级裁判池：连续两个赛季评分后10%者，降入B级。 - B级裁判池：连续两个赛季评分后10%者，降入C级。 - C级裁判池：连续两个赛季评分后20%者，取消注册资格，三年内不得重新报考。 **选拔**：任何年满18岁的中国公民均可报名参加年度裁判资格考试，通过者从C级逐级晋升。 **淘汰**：每场比赛后由裁判监督（30%）、两队队长（20%）、联赛技术委员会（30%）、实名制注册球迷投票（20%）四方独立打分（权重30%、20%、30%、20%），评分即时上链，不可篡改。 连续两个赛季评分末位者，降级或取消注册资格。 ### 5. 争议判罚评议机制 - **法理依据**：严格以[国际足球协会理事会（IFAB）《足球竞赛规则》](https://www.theifab.com/laws-of-the-game-documents/)为唯一标准。 - **常规评议**：赛后48小时内，由联赛公司评议部门对接中国足协，针对争议判罚组织非利益相关方的裁判员、教练员、退役或现役球员组成10人评议组，对争议判罚进行复核，并由足协发布权威报告。 - **重大争议多元投票机制**：当争议判罚直接影响升降级结果、引发全国性舆论关注或常规评议组无法裁定其性质时，启动多元投票机制。投票结果作为裁判考核与惩处的核心依据，但**比赛结果不予更改**。 - 全国实名注册球迷投票：结果占 **20%** - 非利益相关方裁判员（随机抽取50人以上）：结果占 **30%** - 各级现役教练员及球员（分级随机抽取）：结果占 **30%** - 体育专家（法学、体育学等）：结果占 **20%** ### 6. 错判漏判的考核与惩处 引发争议并经确认的误判，加倍计入裁判扣分权重。根据赛季累计误判次数及严重性，执行停哨、降级等阶梯式处罚。 ### 7. 薪酬保障 A级裁判实行年薪制，薪酬对标A组球员平均年薪的50%以上。B级裁判实行高额单场津贴加基本工资。C级裁判按场次领取津贴。所有薪酬发放记录上链，透明可查。 ## 七、主教练体系：市场化聘任，成绩倒逼，梯队强制 ### 1. 资质认证与全球开放 向全球开放教练注册，任何持有国际足联或各大洲足联认可教练证书者均可执教。同时建立国内教练培训学院，设立从C级到职业级的完整培训体系。 ### 2. 聘任完全市场化 各球队主教练的选拔、聘任、薪资，完全由球队自身决定，联赛公司不予干预。 ### 3. 与升降级绑定的契约条款 主教练合同必须包含： - **降级解约条款**：A组球队降入附加赛池并最终失去全国联赛资格，主教练合同自动解除，不设违约金。 - **升级续约条款**：B组球队取得前4名直升A组，主教练自动触发一年续约选项，薪资按约定比例上调。 - **附加赛责任条款**：须参加附加赛的球队，主教练赛季评估中附加赛成绩权重占70%以上。 ### 4. 淘汰机制：数据驱动的公开评估 赛季末，联赛公司技术委员会发布《赛季教练评估报告》，基于球队积分与预期积分的差值、年轻球员出场时间占比、比赛内容客观数据等指标。连续两个赛季积分差值在同级联赛教练中排名末位者，触发公开的“建议更换教练函”。 ### 5. 梯队强制绑定 每支A组球队主教练有法定义务每个赛季为U19梯队提供不少于200小时的技术指导，并提交详细的梯队评估报告。这是主教练续约的必要条件之一，确保顶级教练资源向下渗透。 全国足球联赛纪检官网举报专区、纪检官方小程序、联赛专属监督邮箱，无第三方跳转入口，全天无休受理线索 2. 线下渠道 各赛区专属举报信箱、纪检专员线下接待点 3. 专线热线 全国统一打假赌黑专属举报专线，专人值守登记 ### 2. 两种举报模式 #### 实名举报（优先核查、重奖激励） 要求：填报真实姓名、身份证、联系方式、所属身份（球员 / 教练 / 裁判 / 球迷 / 业内从业者） 权益： 1. 线索优先加急核查，全程专人对接进度 2. 查实案件后，按涉案金额 / 赛事影响层级发放高额实名举报奖金 3. 严格保密个人信息，严禁泄露给涉事方，杜绝打击报复 4. 提供人身安全报备通道，遇威胁可申请联赛方协助报备安保与公安 #### 匿名举报（无身份压力，全民监督） 要求：无需填写任何个人信息，仅提交时间、场次、人员、证据、违规细节 规则： 1. 仅作线索储备，优先级次于实名线索 2. 无现金奖励，仅作为舆情及排查依据 3. 可预留私密留言暗号，方便后续补充证据 ### 3. 举报必备提交材料（硬性标准，提高核查效率） 基础信息：赛事场次、对阵队伍、比赛时间、涉事人员名单 实证材料：聊天记录、转账凭证、录音录像、下注凭证、口头证词笔录 违规事实：明确写明假球操盘、默契球、收受贿赂、操控比分、裁判吹黑哨等具体行为 ### 4. 线索处置流程（闭环流程） 登记归档：纪检组专人统一收录所有举报线索，分类建档 初审筛查：72 小时内完成初步甄别，剔除恶意诬告、不实谣言 立案核查：有效线索立即成立专项调查组，同步启动前文即时管控措施（停赛、冻结分红、托管球队） 结果公示：查实 / 辟谣结果统一对外公示，做到公开透明 ### 5. 配套约束机制 #### 诬告追责 恶意捏造事实、虚假举报蓄意抹黑球员 / 球队 / 裁判，一经查实，视情节处以行业通报、罚款，情节严重追究法律责任，终身限制足球相关举报及观赛从业权限 内部检举优待 队内球员、教练、工作人员主动内部自首举报，可从轻减免自身处罚，从轻处置 #### 保密铁规 所有举报信息仅限纪检核查人员查阅，泄露举报人信息者，直接依规重处 ### 6. 衔接处罚条例 所有经举报查实的假赌黑行为，直接无缝对接你之前制定的： 人员终身禁业、球队托管、系统性违规全队解散、移送公安刑事追责全套惩处规则，形成举报 - 核查 - 管控 - 重罚完整闭环。 - **即时措施**：一旦查实，涉嫌人员立即停职/停赛；冻结涉事俱乐部在公共池中的全部分红份额；联赛公司派驻管理小组托管俱乐部，确保球队正常参赛。 - **调查结论后处置**： - **个人**：确认参与假赌黑的裁判员、教练员、球员，终身禁止从事任何足球相关活动，通报全国并报国际足联备案。 - **俱乐部**：确认为系统性行为的，**整支球队就地解散**，取消联赛资格。该省参赛席位由省内联赛冠军递补。俱乐部全体管理人员列入足球行业终身禁足黑名单。 - **司法衔接**：联赛公司须主动向公安机关移送全部证据材料，依法追究相关人员刑事责任，并提起附带民事诉讼，追回全部经济损失。 兼顾国内各地区足球发展不均衡现状，实行差异化梯队管控模式，顶层重底蕴筑基，下层重练兵育才，精准适配两级联赛定位，聚焦青年人才培育，打通本土新星成长通道。 ### 1. 准入红线：A组强制达标，B组全面放开 #### A组（顶级16支省级代表队） 执行硬性准入审核制度，必须配齐U19、U17、U15三级完整青训梯队，人员编制、专业训练场地、持证教练团队、日常训练及内部赛事体系全部验收合格。赛季赛前资质审核未达标的队伍，直接取消本赛季联赛参赛资格，空缺参赛席位由该省级行政区省内顶级联赛冠军队伍顺位递补参赛。 #### B组（次级18支省级代表队） 不设立任何梯队建设强制考核指标，无硬性建设任务。各省可依据自身足球底蕴、财政预算、人才储备自主规划青训发展节奏，自由选择建设规模与进度，不做统一硬性约束，核心侧重青年球员实战锻炼。 #### 升降级梯队联动规则 - B组队伍凭借联赛排名成功升入A组后，立刻启用A组全套梯队准入硬性标准； - 升级队伍需在官方规定时限内补齐三级梯队并完成全项资质核验； - 逾期无法达标、审核未通过者，直接取消本赛季升级资格，由联赛同组别下一名次队伍依次替补晋级； - 按排名顺延递补，直至凑齐16支全部符合梯队准入标准的A组参赛队伍。 #### 配套补充细则 - 每年联赛开赛前期统一复核所有A组队伍梯队资质，动态清退不合格队伍，稳固顶级联赛青训底蕴； - 从A组降级至B组的队伍，自动解除三级梯队强制建设要求，回归自主自由建设模式，专注青年球员培育； - 顶层夯实竞技实力，下层全力培育新人，分层实现不同发展目标，推动全国足球人才均衡发展。 ### 2. 全国统一梯队联赛体系 - 全国U19、U17青少年梯队联赛，与成年组省级联赛赛程同步开展，全国34支省级队伍全员参与，实行统一主客场赛制，以赛促练提升青年球员实战能力； - U15梯队选材严格扎根校园足球体系，球员全部从本省中学生足球联赛、校园足球各级赛事中择优选拔；省级中学生联赛前八名队伍核心球员，自动获得本省省队U15梯队正式试训资格，打通校园与职业青训衔接通道。 ### 3. 学籍保留+多元化升学保障通道 联合教育主管部门出台配套扶持政策，彻底打消球员与家长后顾之忧，助力青年球员全方位发展。 - 所有在册梯队球员全程保留原有在校学籍，专业集训、外出赛事时长可统一折算学业学分，确保踢球不耽误学业； - 年满18周岁未走上职业足球道路的梯队球员，符合相关评定标准，可享受全国统一高考体育类加分政策，或通过体育单招途径免试、降分进入合作公办高校就读； - 打通“体教融合”双向出路，实现职业发展与学业提升并行，保障球员退役后仍有稳定升学、就业路径。 ### 4. 青年球员强制出场规则 #### A组成年一线联赛 侧重竞技成绩与联赛观赏性，无硬性年轻球员出场强制要求，队伍可自由调配阵容，全力冲击赛事名次、争夺联赛荣誉，同时鼓励自主启用青年球员锻炼队伍。 #### B组次级联赛 核心定位为青年人才练兵阵地，聚焦新人培育，明确强制出场要求，鼓励多启用年轻球员实战历练： - 硬性规定：每场比赛首发阵容至少安排2名U21球员，确保青年球员获得稳定首发实战机会； - 正向激励：超额启用U21及以下年轻球员、大力挖掘本土新星的队伍，赛季末给予青训积分加成、联赛分红倾斜、升级名额优先推荐等多重奖励； - 约束处罚：未达标队伍扣除部分联赛福利，连续两赛季不落实新人练兵要求，限制其后续冲击升级资格。 #### 省内地方联赛 作为青年人才培育的基层阵地，强化年轻球员启用力度，统一硬性要求：每支队伍首发阵容至少安排4名U21球员，下沉基层持续夯实青年人才储备根基，为省级队伍输送优质后备力量。 ## 十、地方特色赛事的体系化并入 省级各地方联赛需充分发挥地方特色，各级联赛下属分公司应积极配合当地足协，共同推动足球市场繁荣发展。同时，需高度重视足球梯队建设，青少年是足球事业的未来，要以地方赛事为载体，搭建青少年足球培养平台，为足球事业长远发展筑牢人才根基。 全国范围内自发兴起的“村超”“苏超”等地方性赛事，是足球运动扎根基层、激发全民参与热情的最生动实践与证明。为进一步激活基层足球活力、完善全国足球体系，本方案将此类地方特色赛事正式纳入全国足球体系，核心原则明确为：**尊重自发，分类接入，不改造，只赋能，充分授权。** ### 1. 充分授权：赛制自定，只看结果 充分赋予地方分公司自主权限，鼓励其主动发挥主观能动性，在“全国一盘棋”、为全国足球事业输送优质人才的大基调下，自主灵活设置赛事赛制。联赛公司仅对最终结果进行考核，具体要求如下： - 各省省内联赛或地方特色赛事的前3名，需与本省B组球队（若该省设有B组球队）开展附加赛，争夺更高层级赛事资格。 - 关于省内联赛及地方特色赛事前3名的产生方式，无需统一标准——无论是通过联赛循环、杯赛淘汰、锦标赛比拼，还是混合赛制开展，均由地方分公司结合本省实际情况、足球发展基础自主决策确定。 ### 2. 分类接入，承认存量 针对不同发展水平、不同影响力的地方赛事，采取分类接入模式，充分承认现有赛事存量成果，不搞“一刀切”，具体分类如下： - **已有全国影响力的赛事（如村超）**：授予“全国地方足球示范赛事”荣誉称号，赋予赛事外卡直通车资格，其原有品牌标识、赛事赛制保持完全不变，充分保留赛事自身特色与生命力。 - **已有稳定运营的城市联赛（如苏超）**：赛事冠军可直接获得省联赛正赛参赛资格，赛事组委会继续负责赛事运营、组织等相关工作，联赛公司提供必要赋能支持。 - **各类业余联赛**：按照地域划分，统一归入省级联赛体系进行管理，联赛公司仅提供资源、技术等方面的赋能支持，不改变其原有运营模式、赛制及组织架构。 ### 3. 五年地方赛事保护期 设立为期五年的地方赛事保护期，保护期内明确以下要求：不强制改变地方赛事的原有赛制、赛事名称及运营主体；地方赛事可根据自身发展需求，自主选择是否接入全国转播体系及全国赞助体系，充分保障地方赛事的自主性与灵活性。 ### 4. 与人才通道无缝衔接 建立地方赛事与人才选拔的无缝衔接机制，地方特色赛事中涌现的优秀球员，将直接进入省队选拔视野，纳入全省足球人才储备库。同时，打通“省内联赛→附加赛→全国联赛B组→A组”的完整人才上升通道，助力优秀基层球员实现从村口赛场到全国联赛的梦想，为全国足球事业输送更多优质后备力量。尤其要注重青少年梯队建设，在地方赛事体系中增设青少年组别，挖掘培养适龄足球人才，鼓励孩子们参与足球运动，夯实足球事业发展的后备力量，因为孩子们才是足球事业的未来与希望。 ## 十一、全面透明制度：向法定监督机构透明，建立申请公示机制 中国足球的信任重建，必须从彻底的透明开始。但透明不等于无边界的信息泛滥，而是建立一套有监督、有程序、可追责的财务公示制度。本方案实行向国家法定监督机构完全透明，同时开放公民与机构申请公示的双轨机制，以制度性透明筑牢信任根基，推动中国足球健康有序发展。 ### 1. 法定透明对象 联赛公司、各地方分公司、裁判公司、各俱乐部的全部财务数据，须遵循“实时、完整、可追溯”原则，无条件向以下国家法定监督机构开放，接受全方位监督核查： - **国家审计署**：作为国家财政监督的最高机关，有权随时对所有主体的全部账目进行审计核查，重点核查资金流向、收支合规性、资产真实性，确保资金使用符合国家法律法规及行业规范。 - **国家体育总局**：作为体育事业的主管部门，有权随时审查所有主体的财务数据、运营流程及相关台账，监督各项政策落实情况，及时发现并纠正运营中的违规问题。 - **国家税务总局**：作为国家税收征管机关，有权随时稽查所有主体的涉税账目，重点核查工资红线税、球员薪酬所得税、俱乐部营业税、增值税等税种的申报、缴纳情况，确保依法足额纳税，杜绝偷税、漏税、逃税等行为。 ### 2. 申请公示制度 为保障公民与社会机构的知情权、监督权，建立“申请\-审核\-答复\-申诉”的闭环申请公示机制，具体要求如下： - 申请主体：任何年满18周岁的中国公民，或在中国境内合法注册的机构（包括媒体、研究机构、律师事务所、行业协会等），均为合法申请主体。 - 申请流程：申请主体需向联赛公司提交书面申请，明确载明查询目的、具体查询内容范围（需符合法律法规及行业规范，不得涉及国家秘密、商业秘密及个人隐私），并提供相关身份证明材料（公民提供身份证复印件，机构提供营业执照等注册证明）。 - 答复要求：联赛公司收到申请后，须在15个工作日内作出书面答复，明确告知是否同意公示、公示的内容及方式；如拒绝公示，必须详细说明拒绝理由（需符合法律规定，不得无故拒绝合法申请）。 - 申诉机制：申请人对联赛公司的拒绝决定不服，或对公示内容有异议的，可在收到答复之日起10个工作日内，向国家体育总局提起申诉，国家体育总局须在20个工作日内作出申诉处理决定，并书面告知申请人。 ### 3. 定期公开报告 为保障公众的基本知情权，即使无任何主体提出公示申请，联赛公司也须履行定期公开义务，在每个赛季结束后60日内，向社会公开发布《年度财务报告摘要》，确保内容真实、准确、完整，主要包含以下核心内容： - 联赛整体收支情况：明确披露联赛总收入（含赞助收入、转播收入、门票收入等）与总支出（含运营成本、人员薪酬、青训投入等）明细。 - 公共池资金分配情况：公开公共池资金总额、资金来源，以及34家地方分公司的分红金额、分配依据、到账情况。 - 工资红线税相关情况：披露工资红线税的征收总额、征收标准、征收流程，以及资金分配明细（如用于青训投入、联赛公益项目等）。 - 青训奖励金发放情况：公开青训奖励金的发放总额、发放标准、发放对象，以及各俱乐部、青训机构的具体领取金额。 - 核心人员薪酬情况：披露联赛公司高管薪酬总额、薪酬构成（基本工资、绩效工资等），以及裁判公司薪酬总额、裁判薪酬发放标准。 ### 4. 技术保障：区块链存储，分层访问权限 为确保财务数据的真实性、不可篡改性，搭建区块链财务数据存储系统，实行“分层授权、分级访问”机制，明确不同主体的访问权限，兼顾监督效率与信息安全： - 最高权限：国家审计署、国家体育总局、国家税务总局拥有区块链系统最高访问权限，可查看链上所有财务数据、操作日志，实现全程可追溯、可核查。 - 临时权限：经申请并获批的公民或机构，将获得临时访问权限，访问范围严格限定在其申请的内容范围内，权限有效期根据查询需求合理设定，到期自动失效。 - 公众权限：普通公众可通过联赛官方平台、国家体育总局官网等渠道，免费查看定期公开发布的《年度财务报告摘要》，无需额外申请。 同时，建立区块链系统安全保障机制，配备专业技术团队，防范数据泄露、篡改、攻击等风险，确保数据安全。 ### 5. 实施意义 透明是信任的前提，也是中国足球重建公信力的关键举措。向法定监督机构完全透明，是对国家法律法规的敬畏，也是对行业规范的坚守，能够有效遏制违规操作、资金挪用、偷税漏税等乱象，确保每一分资金都用在足球事业发展上。 保障公民与机构的合法查询权，是让群众参与监督、让权力在阳光下运行的重要途径。只有让群众看得见、查得到、能监督，才能逐步重建群众对中国足球的信任——家长才敢把孩子送到绿茵场上，培养足球梦想；社会才愿意投入资源，支持足球事业；每一个怀揣足球梦想的孩子，才能拥有公平、公正的发展环境。 需要明确的是，透明不是无边界的信息轰炸，也不是无底线的隐私泄露，而是一种有秩序、有程序、有边界的制度性敬畏。当每一分钱都在法定监督下规范流转，当每一项决策都能接受公众监督，当每一个违规行为都能被及时发现、严肃追责，中国足球才能真正取信于民、取信于社会，才能夯实发展根基，实现长远健康发展。 ## 十二、保险与风险保障体系 ### 1. 强制险种全覆盖 整个联赛体系内的所有参赛主体和人员，均须强制参保。覆盖范围包括： | 参保主体 | 险种要求 | | -------------------------------- | -------------------------------------- | | 所有注册球员（含梯队、地方赛事） | 运动意外伤害险、重大疾病险、职业伤残险 | | 所有注册裁判员 | 职业责任险、运动意外伤害险 | | 所有注册教练员 | 运动意外伤害险 | | 所有赛事主办方 | 赛事公众责任险、赛事取消险 | | 所有比赛场馆 | 场馆责任险 | ### 2. 统一招标，全国统筹 由联赛公司作为统一投保人，面向全国保险机构进行年度招标。34个省的所有赛事、所有球员、所有裁判和教练，打包进入同一个保险池。这种规模效应能获得最优费率，同时确保欠发达地区获得与发达地区完全相同的保险保障。 ### 3. 纳入银保监会监管 所有涉及本联赛体系的保险产品、保费收入、出险赔付，均须纳入中国银行保险监督管理委员会的监管范围。保险产品的条款和费率须经银保监会备案或审批，承保机构的偿付能力须符合监管标准，所有出险理赔记录须向银保监会报备。 ### 4. 保险收入纳入总资金池 联赛公司设立**联赛自保险基金**，各参赛主体缴纳的保费首先进入该基金。小额赔付由自保险基金直接支付，大额赔付由商业再保险公司承担。赛季末，自保险基金的结余部分，纳入联赛总收入公共池，与其他收入一同按1/34分红。如出现亏损，由公共池资金优先补足，确保球员保障不受影响。 ### 5. 出险赔付的透明化 所有出险申请、赔付决定、赔付金额，均记录上链，不可篡改。球员或其监护人可随时通过公开查询系统查看自己的保险状态和理赔进度。 ## 十三、球队欠薪解散处置机制 ### 1. 总资金池的兜底功能 当任何一支联赛体系内的球队发生欠薪或解散事件时，联赛公司须在30日内启动应急处置程序，从总资金池中调拨专项资金，用于垫付所欠薪资。垫付范围为球员、教练员及工作人员的合法合同薪酬，不包括奖金、签字费等非固定收入。 ### 2. 球员合同托管与挂牌转让 欠薪解散时，受影响球员的合同由联赛公司暂时托管，球员不得自行与新俱乐部签约。联赛公司在30日内将受影响球员列入“特殊转会名单”，面向所有联赛体系内俱乐部公开挂牌。新俱乐部与联赛公司协商转会费，转会费基于球员核定身价、剩余合同总额及球员本人意愿确定。 ### 3. 转会费分配顺序 转会费到账后，按以下优先级分配： | 优先级 | 用途 | 说明 | | -------- | ------------------------ | ------------------------------------------------ | | 第一顺位 | 清偿球员被欠薪资 | 法定优先权，任何机构不得截留 | | 第二顺位 | 清偿教练组及工作人员欠薪 | 保障所有劳动者权益 | | 第三顺位 | 剩余部分归入总资金池 | 用于偿还资金池此前垫付的欠薪款项 | | 第四顺位 | 仍有剩余 | 归入解散球队所在省的地方分公司，专项用于该省青训 | ### 4. 无人问津球员的处理 若挂牌90日后仍无俱乐部接手，球员合同正式解除，转为自由球员。球员被欠薪资由总资金池全额垫付。总资金池获得对该解散球队及其实际控制人的代位追偿权。该球员在解除后24个月内再就业的，新俱乐部须向总资金池支付一定比例的补偿金，以回填垫付资金。 ### 5. 性质分级与追偿惩罚 垫付比例根据事件性质，由联赛公司董事会联合国家体育总局进行调查后裁定。对恶意欠薪方，联赛公司将其列入联赛永久黑名单，其关联方终身不得以任何形式参与本联赛体系的投资、管理或运营，并依法追究其刑事责任。 ### 6. 财务衔接 欠薪垫付作为总资金池的法定支出项，优先级高于赛季末均摊分红。所有垫付记录、追偿进度、惩罚决议均上链公开，接受国家审计署和国家体育总局的实时监督。 ## 十四、改革预期与社会效应 1. **全民动员，打通人才壁垒**：当每个省份都有了自己的代表队，足球将不再是几个大城市的游戏。从校际赛到县区赛，从村超到省级联赛，从B组到A组，一条完整的人才筛选金字塔将首次贯通全国。一个西藏玉树的孩子，可以在家门口踢出名堂，代表本省征战全国。 2. **市场激活，创造全新产业体量**：省级行政单位背后的政治、经济与情感资源将被全面调动，创造出一个覆盖全国的超级体育商业市场。34支球队成为34个可估值、可交易的独立IP，自由转会与培训补偿制度让青训成为一门可持续的生意，球探、经纪等外围商业机构蓬勃发展，催生多层次的体育资本市场。 3. **文化重塑，以透明赢回信任**：用残酷且公平的升降级制度倒逼竞争，用向法定机构透明与申请公示制度重建公信，用“先富带动后富”的均摊机制凝聚全国力量，用独立职业化的裁判体系守护公正，用强制梯队建设夯实未来根基，用完善的保险与欠薪处置机制保护每一位从业者，用开放包容的地方赛事通道连接每一个村庄的足球梦想。 4. **制度创新，探索体育治理新范式**：足协回归服务角色、联赛公司精英治理与加权否决权、法定机构监督与公民申请公示相结合的透明制度、总资金池兜底的风险保障体系——这套权力与运行体系的设计，不仅服务于足球，更可能为中国体育治理的整体改革提供一个可参照的样本。 --- 此方案是一次系统性的顶层重构。它可能在初期面临强队与弱队之间的实力断层，但正是这种断层，将成为推动落后地区跨越式发展的最大动力。它并非修修补补的改良，而是以全国为视野、以人民为根基、以法治与透明为灵魂的彻底重建。 --- （全文完）
